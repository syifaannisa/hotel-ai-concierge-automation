# Project Overview

## Hotel AI Concierge Automation

A modular hotel guest-service automation built with n8n, Google Gemini, Telegram, Gmail, Tally, and Google Sheets.

## Business problem

Hotel teams often manage repetitive guest enquiries across disconnected channels. This creates slow responses, repeated administrative work, inconsistent answers, and limited visibility.

## Solution

The project separates each guest channel into an independent n8n workflow while using a shared operational data layer.

### Implemented demonstration workflows

- Website and Tally guest intake
- Telegram concierge with conversation state
- Email concierge with language and intent detection

### Planned integrations

- Twilio SMS
- WhatsApp Cloud API
- Production hotel PMS

## What this project demonstrates

- business-process analysis;
- modular workflow architecture;
- API and AI integration;
- conversational state management;
- structured data handling;
- secure public configuration;
- technical documentation and handover thinking.

## Important scope note

Hotel Adriatica is a fictional demonstration property. The repository is a portfolio project and does not claim production deployment.
