# Configuration Reference

## Environment variables

| Variable | Required | Purpose |
|---|---:|---|
| `GEMINI_API_KEY` | Yes | Authenticates Google Gemini requests |
| `TELEGRAM_BOT_TOKEN` | Telegram only | Authenticates Telegram Bot API calls |
| `GOOGLE_SHEET_ID` | Yes | Identifies the shared workbook |
| `GOOGLE_SHEET_URL` | Optional | Convenience reference for URL-based nodes |
| `PMS_BASE_URL` | Future | Base URL for a production hotel PMS |
| `PMS_API_KEY` | Future | Authentication for a production PMS |
| `POSTMARK_API_TOKEN` | Optional | Alternative transactional email provider |

## n8n credentials

The public JSON files intentionally do not contain local n8n credential metadata.

After import, select:

- Google Sheets OAuth2;
- Gmail OAuth2.

## Channel configuration

### Tally

Confirm that the form labels match the labels expected by the normalisation node.

### Telegram

Configure Telegram to send updates to the production webhook generated by the imported n8n workflow.

### Gmail

Review inbox filters before activation to prevent the workflow from processing unrelated messages.
